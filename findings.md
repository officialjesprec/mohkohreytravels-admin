# Admin Dashboard Findings

- **Tech Stack**: Vite, React 19, Tailwind CSS v4, Lucide React, Sonner (Toasts), Motion (Animations).
- **Styling Rules**: 
  - Primary Font: Montserrat (Headings).
  - Body Font: Plus Jakarta Sans / Inter.
  - Core Brand Color: #1E293B (Primary Blue/Slate), #FFA000 (Gold).
- **Design Observations**:
  - The Admin layout uses a Sidebar (`w-64`) with a Sticky Header (`h-20`).
  - Colors were updated from boilerplate blue to the Mohkohrey Travels corporate palette.
  - Assets imported: `logo.png`, `favicon.png`.
- **Feature Gap**: 
  - Currently has: Blog, Tours, Bookings, Gallery, Settings.
  - Missing: Visa, Passport, Study Abroad specific dashboards.
