# Tech Stack Requirements

- **Frontend**: React, Next.js 15 (App Router).
- **Styling**: Tailwind CSS (Mandatory).
- **UI Components**: shadcn/ui.
- **Backend/Database**: Supabase.
- **Authentication**: NextAuth.js (or Supabase Auth if preferred by PRD).
- **Images**: Cloudinary.
- **Performance**: Lighthouse score of 90+ for Mobile and Desktop. Faster loading via Edge Delivery (Vercel).
